# 实践沉淀模板

## 为什么需要这份模板

学习 agent 最怕只停留在“看懂了一点点”，但没有沉淀。

这份模板是为了把后续每次学习和实践都补回仓库，变成别人也能直接复用的内容。

## 推荐记录的内容

### 1. 这次在学什么

例如：

- runtime
- tool use
- permission system
- multi-agent orchestration

### 2. 这次原来哪里没想清楚

例如：

- 我把 runtime 和 Node 混了
- 我以为 agent 就是模型
- 我以为多 agent 就是并行

### 3. 这次修正后的理解

用最短的话写清楚：

- 现在我怎么理解这个概念
- 最好配一个比喻

### 4. 对应到哪个源码位置

至少记 1 到 3 个关键文件。

### 5. 如果自己做 agent，这次学到什么

例如：

- 我以后做 tool system 时也要先设计权限
- 我以后不要把 command 和 tool 混在一起
