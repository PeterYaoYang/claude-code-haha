# Docs

## 这套文档是什么

这不是一组零散笔记，而是一套围绕 `claude-code-haha` 搭建的中文学习文档。

目标是帮助后来者：

- 先建立 agent 的核心概念
- 再理解 Claude Code 为什么强
- 再按路线读源码
- 最后把自己的学习和实践沉淀回仓库
